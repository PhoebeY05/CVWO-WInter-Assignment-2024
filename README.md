## Dependencies
- Ruby 3.3.6
- Rails 8.0.0
- node v22.12.0
- npm 10.9.0 OR yarn 1.22.22
## Local Setup
1. Download the files
```
git clone https://github.com/PhoebeY05/CVWO-Winter-Assignment-2024.git
```
2. Install dependencies
```
bundle install
npm install
```
3. Setup SQLite3 database
```
rails db:migrate
```
4. Start the server
```
bin/dev
```
